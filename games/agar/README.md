# Agar

An Agar.io-inspired multiplayer game where players control cells and compete to become the largest.

## Game Mechanics
- Move cell with mouse
- Absorb smaller cells to grow
- Split to catch faster opponents
- Eject mass strategically
- Largest cell at time limit wins

## Tech Stack
- Phaser 3 for game engine
- Socket.IO for multiplayer sync
- Custom physics for cell mechanics
- TypeScript for type safety
