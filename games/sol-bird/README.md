# Sol Bird

A Flappy Bird-inspired multiplayer game where players compete to fly the furthest through pipe obstacles while collecting SOL coins.

## Game Mechanics
- Tap/click to flap and stay airborne
- Navigate through pipe gaps
- Collect SOL coins for points
- Compete against other players in real-time
- Last player standing wins the pot

## Tech Stack
- Phaser 3 for game engine
- Socket.IO for multiplayer sync
- TypeScript for type safety
