# Slither

A Snake.io-inspired multiplayer game where players grow their snakes and compete for dominance.

## Game Mechanics
- Control snake with mouse/keyboard
- Eat pellets to grow longer
- Trap other players to eliminate them
- Absorb eliminated players' mass
- Biggest snake at time limit wins

## Tech Stack
- Phaser 3 for smooth gameplay
- Socket.IO for multiplayer sync
- Matter.js for physics
- TypeScript for type safety
